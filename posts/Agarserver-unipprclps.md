## Universal Paperclips Portable & The Game

I mentioned that I would be making an idle game, and it will be inspired by Universal Paperclips. In the meantime, however, I figured that Universal Paperclips, being a game, needed a portable non-web version. This allowed me to experiment with Electron. Electron is basically a browser emulator, which I was able to package into an executable using Electron-packager. I don't think I would even call this a project, as it required little to no code edits and took only one day. It did however, give me a learning opportunity of packaging web development languages and making portable apps out of it. I have decided that I will program my idle game with HTML, CSS, and JavaScript. I would rather use Python as it is a language I am heavily familiar with, but I think using PyQt is too overcomplicated, as well as Pyside6 I think it's called. I would use Pygame with Python, but it isn't conventional for some of the graphics I would like in the game. I think Pygame's biggest strength is user input, like keyboard and mouse tracking, etc, but I isn't conventional for buttons and other visual elements with a Universal Paperclips-like fashion. That's about it for the game and Universal Paperclips.
